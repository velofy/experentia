# Sample 001: primeintellect.ai (July 2026)

Genre: the contemporary AI-lab landing page. Dark cinematic minimalism with terminal affordances. Deconstructed from a live screenshot (hero viewport, desktop).

## What the page does

| Region | Treatment |
| --- | --- |
| Announcement bar | One mono line on near-black: "Announcing our $130m Series A" with an external-link arrow. News as quiet ticker, not banner. |
| Nav, left | Wordmark pairs a geometric mark with "PRIME" in caps and "Intellect" in an italic contrast face. |
| Nav, center | Four numbered destinations in uppercase mono inside hairline boxes: TRAINING 01, INFERENCE 02, COMPUTE 03, RESEARCH 04. The numbering states "this is a stack with an order", structure as information. |
| Nav, right | Utility links (DOCS, BLOG, CAREERS with a live count "24"), then BOOK A CALL, boxed LOGIN, and a filled white START TRAINING button. Exactly one filled element in the whole bar. |
| Hero image | One full-bleed cinematic photograph: a jungle ravine with a glowing monolith. All emotion is delegated to a single image; the UI stays typographic. Nature-meets-artifact is the brand metaphor. |
| Hero copy | Mono eyebrow THE OPEN SUPERINTELLIGENCE STACK, then a large humanist sans headline "Own Your Intelligence" (sentence case, quiet confidence), then one two-line paragraph, then primary + outlined buttons. |
| Terminal affordance | A literal shell line with a block cursor: `$ pip install prime`. Developer-nativeness proven, not claimed. |
| Credibility strip | "Backed by" followed by investor and researcher names separated by slashes (funds, chip companies, famous individuals). Names as social proof, no logos needed. |
| Case-study rail | A bordered grid of partner cells (ramp, NVIDIA, zapier, character.ai, Goodfire), each with a mono uppercase CASE STUDY / READ MORE tag and arrow. Logos live in cells with hairline borders, not floating in a soup. |

## The genre's rules (reusable)

1. Ground is near-black; type is off-white; there is at most one filled accent element per viewport (here: the white CTA).
2. Uppercase mono microlabels with wide tracking carry ALL wayfinding; the display face carries only the headline.
3. One cinematic image per page does the emotional work. Everything else is flat.
4. Numbers are used honestly: nav order, careers count. Metadata as texture.
5. Proof is textual and specific: an installable command, named backers, named customers.
6. Boxes are hairline, corners are square or barely rounded, nothing has a shadow.

## Tension with the evidence base

High prototypicality WITHIN its genre (every serious AI-lab site now looks like this), low visual complexity in the UI layer, with the image carrying controlled complexity. Consistent with the low-VC high-PT rule from `../research/style.md`. The dark ground trades some broad-audience warmth for in-group credibility, which is an audience parameter, not a universal.

## Token sketch

ground #0A0B0A to #10120F family, ink #F2F1EC, muted #9BA09A, hairline #2A2D2A, one CTA surface #FFFFFF with ink text, mono stack for labels, humanist sans for display, optional italic serif for wordmark contrast.
